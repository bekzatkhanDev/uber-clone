## 1. ОБЩАЯ ХАРАКТЕРИСТИКА ПРОЕКТА

### 1.1 Название проекта
Мобильное приложение для заказа такси (Uber Clone)

### 1.2 Цель проекта
Разработка кроссплатформенного мобильного приложения, реализующего основной функционал сервисов заказа такси для пассажиров и водителей.

### 1.3 Технический стек
| Компонент | Технология |
|-----------|------------|
| Фреймворк | React Native 0.74+ |
| Сборщик | Expo SDK 51 |
| Язык программирования | TypeScript 5.0+ |
| Управление состоянием | Zustand |
| Маршрутизация | Expo Router v3 |
| Стилизация | NativeWind (Tailwind CSS для React Native) |
| Карты и геолокация | Google Maps Platform |
| Интернационализация | i18next |
| Бэкенд | Отдельный сервис на FastAPI |
| Сборка | Gradle (Android), Xcode (iOS) |

### 1.4 Поддерживаемые платформы
✅ Android 8.0+
✅ iOS 15.0+
✅ Веб-версия (экспериментальная)

## 2. АРХИТЕКТУРА ПРОЕКТА

### 2.1 Структура директорий
```
uber-clone/
├── app/                    # Файловая маршрутизация Expo Router
│   ├── (auth)/            # Группа экранов аутентификации
│   │   ├── welcome.tsx
│   │   ├── sign-in.tsx
│   │   └── sign-up.tsx
│   ├── (root)/            # Основные экраны после авторизации
│   │   ├── home.tsx
│   │   ├── find-ride.tsx
│   │   ├── confirm-ride.tsx
│   │   └── book-ride.tsx
│   └── (driver)/          # Раздел для водителей
├── components/            # Переиспользуемые UI компоненты
├── hooks/                 # Кастомные React хуки
├── store/                 # Глобальное состояние приложения
├── lib/                   # Вспомогательные библиотеки и утилиты
├── constants/             # Константы и конфигурации
├── i18n/                  # Локализация (RU, KK, EN)
├── assets/                # Ресурсы (иконки, шрифты, изображения)
└── types/                 # TypeScript интерфейсы и типы
```

### 2.2 Архитектурный подход
Проект реализует архитектуру **Feature-Sliced Design** адаптированную под React Native:
- Разделение ответственности по слоям
- Инверсия зависимостей
- Модульность компонентов
- Отделение бизнес-логики от UI

## 5. РАЗРАБОТКА И ДЕПЛОЙ

### 5.1 Окружение разработки:
- Node.js 18+
- npm / pnpm
- Android Studio / Xcode
- Expo Go для тестирования на физических устройствах

### 5.2 Процесс сборки:
```bash
# Установка зависимостей
npm install

# Запуск разработческого сервера
npx expo start

# Сборка для Android
npx expo prebuild --platform android
cd android && ./gradlew assembleRelease

# Сборка для iOS
npx expo prebuild --platform ios
```
